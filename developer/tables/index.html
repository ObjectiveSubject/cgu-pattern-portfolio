Tables

Use the .table class to assign the default table styles.

Example: Basic table

Expenses
Fees $1,847 per unit
Total Tuition Cost* $110,820
Room & Board $8,100 
<table class="table">
    <tbody>
        <tr>
            <th>Expenses</th>
            <td></td>
        </tr>
        <tr>
            <td>Fees</td>
            <td>$1,847 per unit</td>
        </tr>
        <tr>
            <td>Total Tuition Cost*</td>
            <td>$110,820</td>
        </tr>
        <tr>
            <td>Room &amp; Board</td>
            <td>$8,100</td>
        </tr>
    </tbody>
</table>
