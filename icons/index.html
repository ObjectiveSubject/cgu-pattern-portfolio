Using icons

Icons utilize the :before pseudo element in the css. So you can simply create an element with an icon class and the respective icon will be attached to that element. Common practice is to provide an empty <span> tag.

Search 
<div class="h2">
    <a href="#"><span class="icon-search"></span> Search</a>
</div>
